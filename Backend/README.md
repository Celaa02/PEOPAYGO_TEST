## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

Overview:

General Description:
This project consists of the development of a complete backend application for payroll management. Users can seamlessly register, login, create, view, update and delete.

Technologies used:

Backend: NestJS Database: mysql (TypeORM).

Main features:

Registration and login:
Implementation of a robust registration and login system using JWT authentication for enhanced security.

Employer and Admin:
The application displays a dynamic main view, showing timesheets generated by clients. Users (client/admin) create, view, update, delete timesheet . 

Timesheet list:
A sophisticated search filter has been developed that allows clients to retrieve their records.

create timesheet:
A method has been developed that allows creation of records for timesheet.

update timesheet:
A method has been developed that allows updating each record for timesheet.

delete timesheet:
A method has been developed that allows deleting timesheet records.

Details:

Application of SOLID principles to improve code readability and maintainability.

Introduction of the authentication and authorization function.

Implementation of a robust architectural design, MVC.

Effective error handling.

Data schema integration.

Modeling:

Users
o Username 
o Email 
o Password 
o createdAt 

Timesheet
o Employee 
o Hour_rate 
o hours 
o total_pay 
o createdAt 
o status 
o id_user
